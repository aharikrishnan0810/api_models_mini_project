# API Models Mini Project

This is a Python-based API models mini project focused on data extraction from various document types. The project contains multiple extractors designed to parse and process information from different sources, including Excel files, PDF papers, and resume documents. It leverages API models and implements various prompting techniques for intelligent data extraction and processing.

## Main Function Points

- **Excel Data Extraction**: Processing data from spreadsheet files.
- **Document Content Extraction**: Parsing content from PDF or text formats.
- **Resume Data Extraction**: Extracting information from CV/resume documents.
- **Prompting Techniques**: Implementing multiple prompting techniques for different extraction scenarios.
- **Data Storage**: Organizing extracted data in a dedicated data directory.

## Technology Stack

- **Python**: Primary programming language.
- **API Models**: For intelligent data processing.
- **Requirements Management**: Using `requirements.txt` for package management.
- **File Handling Libraries**: For Excel, PDF, and text processing.
